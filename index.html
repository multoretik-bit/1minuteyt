Получить API-ключ

Сколько канал зарабатывает за 1 минуту видео?

Вставь ссылку на YouTube-канал — посчитаем суммарные просмотры, оценим доход и выведем средний заработок на каждую минуту контента.

  1. Открой console.cloud.google.com и создай проект (бесплатно).
  2. В библиотеке API найди и включи YouTube Data API v3.
  3. Зайди в раздел Credentials → Create Credentials → API key.
  4. Скопируй ключ и вставь в поле выше. Ключ хранится только в твоём браузере и никуда не отправляется, кроме googleapis.com.
